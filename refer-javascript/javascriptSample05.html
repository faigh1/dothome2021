HTML 태그 검색하기

  • <a> 태그는 다른 페이지 이동을 설정합니다.
  • <abbr> 태그는 줄임말이나 머리글자를 표현하는데 사용합니다.
  • <acronym> 태그는 줄임말이나 머리글자를 표현하는데 사용합니다. 현재는 사용하지 않습니다.
  • <address> 태그는 웹 페이지의 연락처 정보를 설정합니다.
  • <applet> 태그는 애플랫 요소를 정의합니다.
  • <area> 태그는 이미지 맵의 영역을 정의합니다.
  • <article> 태그는 문서내에 독립적인 컨텐츠를 정의합니다.
  • <aside> 태그는 문서내에 사이드 컨텐츠를 정의합니다.
  • <audio> 태그는 오디오 파일을 정의합니다.
  • <b> 태그는 글씨를 두껍게 표현할 때 사용합니다.
  • <base> 태그는 문서의 모든 URL의 기본이 되는 기본 URL을 설정합니다.
  • <basefont> 태그는 문서의 기본 폰트, 사이즈, 종류를 설정합니다.
  • <bdi> 태그는 텍스트의 출력 방향을 브라우저가 판단해 표현합니다.
  • <bdo> 태그는 텍스트의 방향을 설정합니다.
  • <big> 태그는 텍스트 크기를 크게해줍니다.
  • <blockquote> 태그는 긴 문장의 인용문을 정의합니다.
  • <blink> 태그는 텍스트를 깜박이게 합니다.
  • <body> 태그는 문서 본문에 해당하는 콘텐츠를 포함하는 영역입니다.
  • <br> 태그는 텍스트를 줄바꿈 할 때 사용합니다.
  • <button> 태그는 클릭 할 수 있는 버튼을 설정합니다.
  • <canvas> 태그는 그림을 그리는 비트맵을 정의합니다.
  • <caption> 태그는 table 요소의 제목을 설정합니다.
  • <center> 태그는 텍스트를 가운데 정렬합니다
  • <cite> 태그는 작품의 제목을 정의합니다.
  • <code> 태그는 컴퓨터 코드를 정의합니다.
  • <col> 태그는 테이블 요소의 행을 나타냅니다.
  • <colgroup> 태그는 테이블 요소의 행을 그룹화합니다.
  • <data> 태그는 주어진 내용을 기계가 읽을 수 있도록 설정합니다.
  • <datalist> 태그는 미리 정의된 옵션의 리스트를 보여줍니다.
  • <dd> 태그는 설명 목록 요소의 설명을 정의합니다.
  • <del> 태그는 문서 내에 삭제된 텍스트를 설정합니다.
  • <details> 태그는 추가정인 정보를 보여주거나 숨기게 할 수 있습니다.
  • <dfn> 태그는 전문적인 용어를 정의할 때 사용합니다.
  • <dialog> 태그는 추가정인 정보를 팝업(박스) 형식으로 보여줍니다.
  • <dir> 태그는 디렉토리 제목을 나열하는데 사용합니다.
  • <div> 태그는 문서의 섹션을 만들거나 영역을 나눌 때 사용합니다.
  • <dl> 태그는 설명 목록을 정의합니다.
  • <dt> 태그는 설명 목록의 용어 및 이름을 정의합니다.
  • <em> 태그는 텍스트를 강조하는데 사용합니다.
  • <embed> 태그는 외부 어플리케이션을 연결합니다.
  • <fieldset> 태그는 폼 요소의 그룹 내부 영역을 설정합니다.
  • <figcaption> 태그는 <figure> 요소의 제목을 정의합니다.
  • <figure> 태그는 그림, 도표, 사진, 코드등의 주석으로 표시합니다.
  • <font> 태그는 폰트의 크기, 색, 종류를 설정합니다.
  • <footer> 태그는 사이트 푸터부분을 정의합니다.
  • <form> 태그는 웹 서버에 정보를 제공하는 폼 컨트롤 유형을 정의합니다.
  • <frame> HTML문서가 표시 될 수 있는 특정 영역을 설정합니다.
  • <frameset> 태그는 프레임을 정의합니다.
  • <h1>~<h6> 태그는 섹션의 제목을 정의합니다.
  • <head> 태그는 사이트의 헤더 요소들을 정의합니다.
  • <header> 태그는 소개 컨텐츠 및 네비게이션 링크를 포함한 섹션입니다.
  • <hgroup> 태그는 제목 요소 그룹화를 정의합니다.
  • <hr> 태그는 문서의 주제를 분리할 때 사용합니다.
  • <html> 태그는 HTML 문서의 루트를 나타냅니다.
  • <i> 태그는 음성이나 분위기 등의 텍스트를 정의합니다.
  • <iframe> 태그는 HTML문서 내에 다른 문서를 포함하는데 사용합니다.
  • <img> 태그는 문서에 이미지를 표현할 때 사용합니다.
  • <input> 태그는 데이터를 입력할 수 있는 폼 컨트롤을 정의합니다.
  • <ins> 태그는 문서 내에 추가된 텍스트를 정의합니다.
  • <kbd> 태그는 키보드 입력을 정의합니다.
  • <keygen> 태그는 보안키를 생성하고 컨트롤합니다.
  • <label> 태그는 <input> 요소의 라벨을 지정합니다.
  • <legend> 태그는 <fieldset> 요소의 영역 제목을 정의합니다.
  • <li> 태그는 목록의 항목을 정의합니다.
  • <link> 태그는 문서와 외부 파일 연결을 정의합니다.
  • <main> 태그는 문서의 주요 콘텐츠를 정의합니다.
  • <mark> 태그는 이미지 맵을 정의합니다.
  • <sample> 태그는 참고 표시를 정의합니다.
  • <marquee> 태그는 텍스트의 스크롤과 움직임을 정의합니다.
  • <menu> 태그는 메뉴 리스트를 정의합니다.
  • <menuitem> 태그는 메뉴 리스트를 항목을 정의합니다.
  • <meta> 태그는 문서에 대한 정보를 제공합니다.
  • <meter> 태그는 범위를 측정하는 게이지 정보를 정의합니다.
  • <nav> 태그는 네비게이션 링크를 정의합니다.
  • <noframes> 태그는 프레임을 지원하지 않는 브라우저를 위한 대체 태그입니다.
  • <noscript> 태그는 스크립트를 지원하지 않는 브라우저를 위한 대체 태그입니다.
  • <object> 태그는 문서내에 포함된 멀티미디어 요소를 정의합니다.
  • <ol> 태그는 순서가 있는 목록을 정의합니다.
  • <optgroup> 태그는 <option> 리스트를 그룹화합니다.
  • <option> 태그는 <select> 요소의 항목을 정의합니다.
  • <output> 태그는 계산의 결과를 보여줍니다.
  • <p> 태그는 문단을 정의합니다.
  • <param> 태그는 <object> 의 매개변수를 정의합니다.
  • <picture> 태그는 디바이스별 이미지를 제공합니다.
  • <pre> 태그는 텍스트의 포맷을 정의합니다.
  • <pregress> 태그는 작업의 진행상태를 정의합니다.
  • <q> 태그는 짧은 인용을 정의합니다.
  • <rp> 태그는 루비 주석을 괄호로 표현하기 위해서 사용합니다.
  • <rt> 태그는 루비 주석에 대한 설명이나 발음을 정의합니다.
  • <rtc> 태그는 루비 주석에 대한 설명이나 발음을 정의합니다.
  • <ruby> 태그는 루비 주석을 정의합니다.
  • <s> 태그는 정확하지 않거나 올바르지 않은 텍스트를 정의합니다.
  • <samp> 태그는 컴퓨터 프로그램의 샘플 출력을 식별하기위한 요소입니다.
  • <script> 태그는 스크립트를 정의하는데 사용합니다.
  • <section> 태그는 주제별 그룹의 콘텐츠 섹션을 정의합니다.
  • <select> 태그는 저작권 또는 법적인 텍스트를 정의합니다.
  • <source> 태그는 미디어 요소에 대한 소스를 지정하는데 사용합니다.
  • <span> 태그는 인라인 요소 그룹화를 정의합니다.
  • <strike> 태그는 취소선을 정의합니다.
  • <strong> 태그는 중요한 의미를 갖는 텍스트를 정의합니다.
  • <style> 태그는 문서에 대한 스타일 정보를 정의합니다.
  • <sub> 태그는 텍스트 아래첨자를 정의합니다.
  • <summary> 태그는 <details>요소 내용에 대한 요약 또는 제목을 정의합니다.
  • <sup> 태그는 텍스트 위첨자를 정의합니다.
  • <svg> 태그는 SVG를 정의합니다.
  • <tbody> 태그는 표의 본문을 그룹화합니다.
  • <td> 태그는 표의 셀을 정의합니다.
  • <td> 태그는 템플릿을 표현합니다.
  • <textarea> 태그는 여러줄의 텍스트를 편집할 수 있는 컨트롤을 정의합니다.
  • <tfoot> 태그는 표의 푸터를 그룹화합니다.
  • <th> 태그는 표의 행이나 열에 머리말을 정의합니다.
  • <thead> 태그는 표의 헤더를 그룹화합니다.
  • <time> 태그는 날짜와 시간을 정의합니다.
  • <title> 태그는 HTML문서의 타이틀을 정의합니다.
  • <tr> 태그는 표의 행을 정의합니다.
  • <track> 태그는 미디어 요소의 트랙을 지정합니다.
  • <tt> 태그는 타자모양의 텍스트를 정의합니다.
  • <u> 태그는 밑줄을 표시할 때 사용합니다.
  • <ul> 태그는 순서가 없는 목록을 정의합니다.
  • <var> 태그는 프로그램 코드에서 변수를 정의합니다.
  • <video> 태그는 미디어 영상을 재생합니다.
  • <wbr> 태그는 줄바꿈 할 위치를 정의합니다.
