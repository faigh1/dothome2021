1부터 N까지의 합 출력하기

Q: 자연수 N이 입력되면 1부터 N까지의 합을 출력하는 프로그램을 작성하시오.

입력 설명
20이하의 자연수 N이 입력됩니다.
출력 설명
1부터 N까지의 합이 출력됩니다.
입력 예제
10
출력 예제
55
{
    function solution(num){
        let answer = Math.ceil(num / 12);
        return answer;
    }
    solution(178);
    console.log(solution(178))
}
